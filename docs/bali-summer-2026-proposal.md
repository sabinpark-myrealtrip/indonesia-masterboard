## 1. 기획 배경 및 목적

### 배경

발리 7·8월은 건기와 성수기가 맞물리는 연간 최대 수요 구간이다. 마이리얼트립 내부 데이터 기준 2026년 여름 발리 수요는 전년 대비 급증하고 있으며, 이 수요를 경쟁사에 빼앗기기 전에 잠재 고객을 전환시키는 것이 목적이다.

### 목적

- 기존 발리 프로모션 페이지(acmBaliSum2026)를 데이터 기반으로 재구성
- CRM 발송을 통해 프로모션 페이지로 랜딩시켜 구매 전환율 제고
- 찜 후 미구매 유저, 예약 리드타임 후기 진입자를 집중 공략

## 2. 현황 진단

### 기존 페이지 문제점

| 항목 | 현재 상태 | 문제점 |
|---|---|---|
| 타이틀 | "발리 전지역 특가 프로모션 전격 오픈!" | urgency 없음. "전격 오픈"은 상시적 표현 |
| 상품 구성 | 나열식 | 큐레이션 근거 없음. 왜 이 상품인지 맥락 부재 |
| 탐색 구조 | 단일 스크롤 | 지역·목적별 필터 없어 이탈 유발 |
| 타겟 분화 | 없음 | 신혼·가족·MZ 각자 원하는 상품이 다른데 동일 노출 |
| 구매 urgency | 없음 | 지금 예약해야 하는 이유 제시 없음 |
| 사회적 증거 | 할인율·별점만 표시 | 찜 수, 리뷰 수 기반 신뢰 구축 미흡 |

#### (3) 체크인 월 기준 YoY 비교

| 체크인 월 | 2025년 | 2026년 | YoY 증감 |

## 4. 개선 전략

### 핵심 방향

> **"전지역 나열"에서 "목적별 데이터 기반 큐레이션"으로**

| 항목 | 기존 | 개선 |
|---|---|---|
| 진입 훅 | "전지역 특가 전격 오픈" | "성수기 전 지금이 가장 저렴" — 83일 리드타임 근거 |
| 탐색 방식 | 나열식 스크롤 | 여행 목적별 탭 + 지역별 탭 |
| 큐레이션 근거 | 없음 | 찜 수 1위~7위 + GMV 파트너 순 |
| urgency | 없음 | 성수기 D-카운트, P25 마감 앵글 |
| CTA | 페이지 내 분산 | 하단 고정 CTA 바 상시 노출 |
